## Usage / Example
### ViT-Lens with InstructBLIP
- Installation:
    1. Install ViT-Lens following the official repo.
    2. Installation for InstructBLIP
        ```bash
        cd vitlens_instructblip/
        pip install -e .
        ```
- Example
    ```bash
    cd vitlens_instructblip/
    python vitlens_instblip_example.py
    ```
